ScratchVerifier Usage

This page contains a walkthrough for using this API.

Basic Walkthrough

The API root URL is https://scratchverifier.ddns.net

After registering, here's how to use this API:

  1. Send an HTTP PUT request to /verify/{username} with the client ID and token you obtained as HTTP Basic authorization
  2. The API will send you a Verification object containing a code
  3. Have your client post that verification code on their own profile comments
  4. Send an HTTP POST request to /verify/{username}
  5. If your client is verified, you should receive a 204 No Content response. Done!
